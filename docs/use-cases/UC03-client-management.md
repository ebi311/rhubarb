# UC03: 利用者管理

## 概要

管理者が訪問介護サービスを受ける利用者の情報を管理する。

## アクター

- 管理者（オフィス管理者）

## 事前条件

- 管理者としてログイン済み

## 基本フロー

### 利用者一覧の確認

1. 管理者が `/admin/clients` にアクセスする
2. 自オフィスの利用者一覧が表示される
3. 契約ステータスでフィルタできる（契約中 / 中断中 / すべて）

### 利用者の新規登録

1. 「利用者を追加」ボタンをクリックする
2. フォームに入力する（氏名、住所）
3. 「保存」をクリックする
4. 契約ステータス `active` で登録される

### 利用者の編集

1. 一覧から利用者の「編集」ボタンをクリックする
2. 氏名、住所を編集する
3. 「保存」をクリックする

### 契約の中断

1. 一覧から利用者の「中断」ボタンをクリックする
2. 確認後、契約ステータスが `suspended` に変更される

### 契約の再開

1. 中断中の利用者の「再開」ボタンをクリックする
2. 契約ステータスが `active` に変更される

## 代替フロー

なし

## 事後条件

- 利用者情報がデータベースに反映される
- 中断中の利用者は基本スケジュール作成時に選択不可

## ビジネスルール

- 削除機能は提供しない（履歴保持のため）
- `suspended` 状態の利用者は新規スケジュール作成時に選択不可
- 既存スケジュールは保持されるが、警告表示される
