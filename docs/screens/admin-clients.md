# 利用者一覧画面

## 基本情報

| 項目 | 値               |
| ---- | ---------------- |
| URL  | `/admin/clients` |
| 権限 | 管理者のみ       |

## 目的

自オフィスの利用者を一覧表示し、追加・編集・契約状態の変更を行う。

## レイアウト

```
┌─────────────────────────────────────────────────┐
│ [ナビゲーション]                                 │
├─────────────────────────────────────────────────┤
│ 利用者管理                    [＋ 利用者を追加] │
├─────────────────────────────────────────────────┤
│ [ステータス: 契約中 ▼]                          │
├─────────────────────────────────────────────────┤

└─────────────────────────────────────────────────┘
```

## 表示項目

### フィルタ

| 項目       | 説明                     |
| ---------- | ------------------------ |
| ステータス | 契約中 / 中断中 / すべて |

### テーブル

| 列         | 説明                          |
| ---------- | ----------------------------- |
| 氏名       | 利用者名                      |
| 住所       | 訪問先住所                    |
| ステータス | 契約中 / 中断中（バッジ表示） |
| 操作       | 編集・中断/再開ボタン         |

## 操作・インタラクション

### 利用者を追加

1. 「＋ 利用者を追加」ボタンをクリック
2. フォームモーダルが開く
3. 氏名・住所を入力
4. 送信で `createClientAction` 実行
5. 成功 → トースト表示、一覧更新

### 編集

1. 行の「編集」ボタンをクリック
2. フォームモーダルが開く（既存データ入力済み）
3. 送信で `updateClientAction` 実行
4. 成功 → トースト表示、一覧更新

### 契約中断

1. 契約中の行の「中断」ボタンをクリック
2. 確認ダイアログ表示
3. 確認後 `suspendClientAction` 実行
4. ステータスが「中断中」に変更

### 契約再開

1. 中断中の行の「再開」ボタンをクリック
2. `resumeClientAction` 実行
3. ステータスが「契約中」に変更
