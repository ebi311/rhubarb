# 基本スケジュール登録画面

## 基本情報

| 項目 | 値                           |
| ---- | ---------------------------- |
| URL  | `/admin/basic-schedules/new` |
| 権限 | 管理者のみ                   |

## 目的

新しい基本スケジュールを登録する。

## フォーム項目

| 項目         | 必須 | 説明                                     |
| ------------ | ---- | ---------------------------------------- |
| 利用者       | ✓    | 契約中の利用者から選択                   |
| サービス区分 | ✓    | サービス区分から選択                     |
| 曜日         | ✓    | 日〜土から選択                           |
| 開始時刻     | ✓    | HH:MM 形式                               |
| 終了時刻     | ✓    | HH:MM 形式（開始時刻より後）             |
| 担当者       | ✓    | スタッフピッカーから選択（許可済みのみ） |
| 備考         |      | 最大 500 文字                            |

## 操作・インタラクション

### 担当者選択

1. 「選択」ボタンをクリック
2. `StaffPickerDialog` が開く
3. 許可済みスタッフのみ表示（利用者・サービス区分でフィルタ）
4. スタッフを選択
5. フィールドに反映

### フォーム送信

1. 必須項目を入力
2. 「登録」ボタンをクリック
3. バリデーション実行
4. 成功 → トースト表示、一覧にリダイレクト
5. 失敗 → フィールド下にエラー表示

### キャンセル

- 「キャンセル」または「← 基本スケジュール一覧」で一覧に戻る

## バリデーション

- 必須項目のチェック
- 終了時刻 > 開始時刻
- 同一スタッフ・同一曜日の重複チェック
- 担当許可リストのチェック
