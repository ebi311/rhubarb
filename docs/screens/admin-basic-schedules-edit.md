# 基本スケジュール編集画面

## 基本情報

| 項目 | 値                                 |
| ---- | ---------------------------------- |
| URL  | `/admin/basic-schedules/[id]/edit` |
| 権限 | 管理者のみ                         |

## 目的

既存の基本スケジュールを編集または削除する。

## フォーム項目

| 項目         | 必須 | 編集可 | 説明                                     |
| ------------ | ---- | ------ | ---------------------------------------- |
| 利用者       | ✓    | ✗      | 変更不可（読み取り専用表示）             |
| サービス区分 | ✓    | ✓      | サービス区分から選択                     |
| 曜日         | ✓    | ✓      | 日〜土から選択                           |
| 開始時刻     | ✓    | ✓      | HH:MM 形式                               |
| 終了時刻     | ✓    | ✓      | HH:MM 形式（開始時刻より後）             |
| 担当者       | ✓    | ✓      | スタッフピッカーから選択（許可済みのみ） |
| 備考         |      | ✓      | 最大 500 文字                            |

## 操作・インタラクション

### 更新

1. 内容を変更
2. 「更新」ボタンをクリック
3. `updateBasicScheduleAction` 実行
4. 成功 → トースト表示、一覧にリダイレクト

### 削除

1. 「削除」ボタンをクリック
2. 確認ダイアログ表示
3. 確認後 `deleteBasicScheduleAction` 実行
4. 論理削除（`deleted_at` 設定）
5. 成功 → トースト表示、一覧にリダイレクト

### キャンセル

- 「キャンセル」または「← 基本スケジュール一覧」で一覧に戻る

## エラーケース

- 存在しない ID → 404 ページ表示
- 削除済みスケジュール → 編集不可、エラー表示
