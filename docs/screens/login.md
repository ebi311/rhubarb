# ログイン画面

## 基本情報

| 項目         | 値                           |
| ------------ | ---------------------------- |
| URL          | `/login`                     |
| 権限         | 未認証ユーザーのみアクセス可 |
| リダイレクト | 認証済みの場合は `/` へ      |

## 目的

ユーザーが Google アカウントでシステムにログインする。

## レイアウト

```
┌─────────────────────────────────────┐
│                                     │
│     ┌─────────────────────────┐     │
│     │                         │     │
│     │    [ロゴ / タイトル]    │     │
│     │                         │     │
│     │  ┌───────────────────┐  │     │
│     │  │ Google でログイン │  │     │
│     │  └───────────────────┘  │     │
│     │                         │     │
│     └─────────────────────────┘     │
│                                     │
└─────────────────────────────────────┘
```

## 表示項目

| 項目                  | 説明                    |
| --------------------- | ----------------------- |
| ロゴ / タイトル       | アプリケーション名      |
| Google ログインボタン | Google OAuth 認証を開始 |

## 操作・インタラクション

### Google でログイン

1. ボタンをクリック
2. Google 認証画面にリダイレクト
3. 認証成功 → `/auth/callback` → `/`
4. 認証失敗 → エラー表示

### エラー表示

- スタッフ未登録の場合: 「このアカウントは登録されていません。管理者に問い合わせてください。」
