# 基本スケジュール一覧画面

## 基本情報

| 項目 | 値                       |
| ---- | ------------------------ |
| URL  | `/admin/basic-schedules` |
| 権限 | 管理者のみ               |

## 目的

週次で繰り返される基本スケジュールを一覧表示し、新規登録・編集・削除を行う。

## 表示項目

### フィルタ

| 項目         | 説明               |
| ------------ | ------------------ |
| 曜日         | 全て / 日〜土      |
| 利用者       | 契約中の利用者一覧 |
| サービス区分 | サービス区分一覧   |

### テーブル

| 列       | 説明                    |
| -------- | ----------------------- |
| 利用者   | 利用者名                |
| サービス | サービス区分名          |
| 曜日     | 日本語表記（月・火...） |
| 時間     | HH:MM - HH:MM           |
| 担当者   | デフォルト担当者名      |
| 備考     | 備考（あれば）          |

## 操作・インタラクション

### 新規登録

1. 「＋ 新規登録」ボタンをクリック
2. `/admin/basic-schedules/new` に遷移
3. フォームに入力して送信
4. 成功 → 一覧に戻る

### 編集

1. 行をクリック
2. `/admin/basic-schedules/[id]/edit` に遷移
3. フォームで内容を変更（利用者は変更不可）
4. 「更新」で送信
5. 成功 → 一覧に戻る

### 削除

1. 編集画面で「削除」ボタンをクリック
2. 確認ダイアログ表示
3. 確認後、論理削除実行
4. 一覧から非表示になる
